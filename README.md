# feedback-widget

Just add the script in your `index.html`
```html
  <script id="feddback-script" type="module" src="./test.js" data-feedback-opts='{ 
    "endpoint": "https://dummy.restapiexample.com/api/v1/create", 
    "id": "dgMaas", 
    "emailField": false, 
    "title": "Feedback", 
    "forceShowButton": true 
  }'></script>
```
